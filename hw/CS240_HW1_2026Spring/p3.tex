\problem{}
Given a set of $n$ points on the real axis $A=\{ A_1,\ldots,A_n\}$ ($A_i\in\mathbb{R}$). Please design
a greedy algorithm to find \textbf{the minimum number of unit-intervals to cover all the given points}. That is to say, each point is covered by at least one interval.

\begin{itemize}
	\item A \textbf{unit-interval} is a closed interval $[l,r]$ where $r-l=1$. 
	\item A point $x$ is \textbf{covered} by an interval $[l,r]$ if and only if $l \leq x \leq r$.
	% \item A point $x$ is \textbf{covered} by a set of intervals if $x$ is covered by at least one interval in the set.
\end{itemize}

\textbf{Example:} 
Given $A = \{ 1.01, 10.1, 3.14, 2.33\}$. Then the minimum number of unit-intervals needed is $3$ and $[1.0, 2.0] \cup  [2.3, 3.3] \cup [9.5, 10.5]$ are $3$ possible unit-intervals covering all points in $A$.

\textbf{Note:} 
Your solution should include the following three parts:
\begin{itemize}
    \item Describe your algorithm in \textbf{pseudocode} or \textbf{natural language}.
    \item Proof of correctness of your algorithm using \textbf{exchange argument}.
    \item Time complexity analysis. Your algorithm should run in $o(n^2)$ time.
\end{itemize}
