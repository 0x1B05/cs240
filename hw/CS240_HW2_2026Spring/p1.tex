\problem{}

Given a sequence of integers \(a_1, a_2, \dots, a_N\), an increasing subsequence is a subsequence \(a_{i_1}, a_{i_2}, \dots, a_{i_k}\), such that
\(1 \le i_1 < i_2 < \cdots < i_k \le N\) and \(a_{i_1} < a_{i_2} < \cdots < a_{i_k}.\) Your task is to find the maximum possible sum among all increasing subsequences of the given sequence.

Note that the longest increasing subsequence does not necessarily have the largest sum. Provide the state transition equation and the corresponding pseudocode.


\textbf{Example 1:}

Input: \(a = [1,7,3,5,9,4,8]\) 

Output: \(18\)

Explanation: One optimal increasing subsequence is \((1,3,5,9)\), whose sum is \(18\).

\textbf{Example 2:}

Input: \(a = [100,1,2,3]\)  

Output: \(100\)

Explanation: Although \((1,2,3)\) is longer, its sum is only \(6\). The subsequence \((100)\) has the maximum sum.
